Angular has two form: reactive vs template-driven

theo angular docs có 4 khác biệt chính giữa 2 form:

# Reactive Template-Driven
Setup sử dụng common component class( formcontrol, formgroup, formbuilder, ..) sử dụng directive( ngModel, ngForm, ...)
DataModel Có cấu trúc(khai báo các thành phần trong form ở file component) Phi cấu trúc
DataFlow Đồng bộ(khi nhập dữ liệu vào thẻ input từ file html giá trị sẽ được cập nhật trực tiếp ở file component thông qua common class) Bất đồng bộ(khi nhâp liệu dữ liệu sẽ được lưu vào trong biến được tạo ở file component, quá trình không giải ra ngay)
Form validation Function(validate ở component) Directive(validate ở template)
Template-Driven

  • import FormModule

  • gồm các directive ngModel, ngForm, ngModelGroup

  • dùng tạo form đơn giản, không có tính tái sử dụng

  • thích hợp làm form đơn giản có thể validate trực tiếp ở template

Reactive

  • import ReactiveFormModule

  • gồm các class formControl, formGroup, formBuilder

  • thích hợp làm form phức tạp
